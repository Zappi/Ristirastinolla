# Ristirastinolla

Aiheena on perinteinen ristinolla, jossa kaksi pelaajaa pelaavat toisiaan vastaan. Pelaajat klikkaavat vuorotellen haluamaansa ruutua ja kun toinen pelaaja on saanut viidensuoran.
